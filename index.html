What are character classes?

symbol equivalent compliance
\ For characters normally treated literally, means that the next character is special and should not be interpreted literally.For characters usually treated in a special way means that the next character is not special and must be interpreted literally.
. [^\n] Metasymbol (if you need to find exactly the point then we escape with a slash: /\./)
\d [0-9] Digital symbol
\D [^0-9] Non-digital symbol
\w [A-Za-z0-9_] Alphanumeric character or underscore
\W [^A-Za-z0-9_] Any character other than an alphanumeric character or underscore
\s [\f\n\r\t\v] Whitespace symbol
\S [^\f\n\r\t\v] Non-whitespace symbol

Anchors:

Anchor Position
^ Beginning of the text
$ End of the text
\B Not a word boundary(inside the word)
\b A word boundary

Quantifier Value

RegExp methods:

Method Value
regexp.exec(str) The exec() method searches for a regular expression match on the specified string. Returns array with results or null.
str.match([regexp]) The str.match(regexp) method looks for matches with regexp in the string str
regexp.test(str) The test() method searches for a regular expression match to the specified string. Returns true or false.
str.search([regexp]) The search() method searches for a match between a regular expression and this String object.
str.split(regexp|substr, limit) Splits a string into an array by delimiter-regexp regexp or substring substr
str.replace(regexp|substr, newSubStr|function[, flags]) The replace() method returns a new string with some or all of the pattern mappings replaced with a substitute
